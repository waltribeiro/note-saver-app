# employee-generator

Application that allows users to:
- create and save notes
- view previously saved notes
- delete previously saved notes

## Technologies
- JEST and Test Driven Development
- Node.js
- Template String Literals
- Audio and Video Editing
- Javascript
- Class Consructors
